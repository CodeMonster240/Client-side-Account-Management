Account registration template

This is an entirely client-side account system. Using JavaScript LocalStorage, it was possible to create an account creation system and login. Check it out!

Login Register Dashboard (Note that this will redirect you to the login page and you'll see why)

Features

  1. Registration
  2. Login
  3. Simple account dashboard
  4. Dashboard security (If the dashboard file is loaded without a redirect from the login page when it will redirect to it.)
  5. Sessions (If the dashboard is reloaded then it will redirect back to login)
  6. Custom messages (incorrect password/email, not logged in, error, session expired)
  7. A cool UI (Credits to CodingNepal for the templates)
  8. Account creation and deletion
